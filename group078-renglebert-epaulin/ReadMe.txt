Arguments : 
    1. Mesh file destination (default : mesh.txt)
    2. Problem file destination (default : problem.txt)
    3. Solver type : (default : B)
        a. B : Band solver 
        b. F : Full solver
        c. G : Conjugate gradient solver
    4. Renumerotation type : (default : X)
        a. 0 : No renumerotation 
        b. X : Renumerotation by X
        c. Y : Renumerotation by Y

All arguments are optional but they must be used in this order.
